02 · Background

결재의 비극 😮‍💨

🧑‍💻직원 — 칼퇴·연차 한 번 내려면 눈치 게임. 죄송한 말투, 장문의 사유, 그리고 반려 공포.
🧑‍💼팀장 — 거절하면 빌런, 다 승인하면 호구. 이러지도 저러지도 못하는 딜레마.
🔁결과 — 서로 눈치만 보다가, 결국 다 같이 참는다.
결재는 원래 ‘설득’인데, 우리는 ‘눈치’로 하고 있었다.
리그오브레거시 · 빼박결재 KNIFEOFF™03
03 · Solution

팀장이 ‘구조적으로’ 거절할 수 없는 결재

🧑‍💻 직원 뷰

  • 목적 선택 — 연차 🌴 / 칼퇴 🔪
  • 질문 몇 개 → AI가 반려 불가능한 근거 5종 자동 생성
  • 대충 쓴 사유를 절절한 읍소문으로 포장 (LLM)
  • 미리보기 → 상신

🧑‍💼 팀장 뷰

  • 상신 즉시 결재함에 대기중으로 도착
  • 승인은 클릭 한 번, 쉬움
  • 거절 버튼은… 구조적으로 불가능
  • 승인 시 承認 도장 + 컨페티 🎉
설득은 자동화, 거절은 불가능.
리그오브레거시 · 빼박결재 KNIFEOFF™04
😱 거절
04 · The Fun

거절을 막는 장치들

🏃

도망가는 거절 버튼

연차 거절 시, 버튼에 다리가 돋아 화면을 뛰어다님. 영원히 클릭 불가 → 결국 승인.

연차 전용
🧮

못 푸는 반려 시험

칼퇴 거절 시, reCAPTCHA “팀장이 맞습니까?” → 풀 수 없는 수학문제 / 없는 고양이 찾기.

칼퇴 전용
✨

AI 읍소문

“그냥 집 가고싶음 ㅋㅋ” → “존경하는 팀장님께 삼가 아뢰옵니다…” 극존칭으로 포장.

🎉

승인의 쾌감

承認 도장이 쾅! 컨페티가 터지고, 나칼퇴 사원은 칼같이 퇴근한다.
